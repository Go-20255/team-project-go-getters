## Progress
1. **Project Structure** 
We've significantly changed the form of the project to better fit our capabilities. Implementing Minesweeper should let us exercise all the same fundamental Go skills as our proposed Critterarium game, while being more reasonable for the time and experience we have.

2. **GUI**
Rendering of game board and objects is functional. `src/game_graphics/` is fully implemented for minimal Minesweeper features. Optimizations and additions might still be needed, but for our MVP this interface meets all requirements.

3. **Game Logic**
A skeleton framework has been stubbed for the game's logic within `src/internal/`. This should give us the right structure to implement our MVP. Additionally, all of our stretch goals should be achievable on top of this framework if time permits.
